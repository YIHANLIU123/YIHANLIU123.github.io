发表于2019-10-21|更新于2023-05-05

说说你对bom 的理解，常见的bom对象你了解那些？

  • bom是浏览器对象模型，提供了与内容与浏览器窗口交互的对象

  • 对象有window、loction、navigator、screen、history

pormise和async awiat 区别是什么?

  • 它俩都是异步请求的方式

  • promise 是es6新增 async 是es7

  • async awiat 是基于promise的 它和promise都是非阻塞的

  • promise 捕获异常的方法是catch async 是try catch

get 和post 的区别

  • get一般是获取数据,post是提交数据用的
  • get的参数会放在url上面,所以安全性比较差,post是放在body中
  • get的请求服务器或者退回是没有影响的,post请求退回时会重新提交
  • get请求会被缓存,post不会
  • get请求都会保存在浏览器历史记录中,post不会

export 与export default 区别？

  • 一个文件的export 可以有多个 export default只能有一个
  • export暴露的是一个接口，在import时不能改变变量名，并且必须加上大括号
  • export default 暴露的是一个变量，在import时可以使用任意变量名导入，不用加大括号

link和import的区别?

加载时间的差异 link在页面载入时同时加载,import页面加载完成后在加载

浏览器兼容性不同: link支持全部浏览器,import不支持ie7 以上的

用法上不同

for in 和 for of 的区别？

forof 和forin 都可以循环数组，forin输出的都是数组的下标，forof 循环出来的是数组每一项得到值

forin 可以遍历对象，forof不能遍历对象

parseint 和 number的区别？

number 可以把字符串转换成数值，也可以转换成浮点数

parseint如果说第一个是数字字符就会转换完成

number转换的时候如果字符里面包含非数字字符就会转换失败

==和===的区别？

它俩的差别就是比较不比较数据类型的问题
